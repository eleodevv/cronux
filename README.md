# Cronux Estático

Esta carpeta contiene las páginas estáticas del sitio web de Cronux.

## Archivos incluidos:

- `index.html` - Página principal del sitio
- `index.css` - Estilos de la página principal
- `documentacion.html` - Página de documentación
- `documentacion.css` - Estilos de la documentación
- `descarga.html` - Página de descargas
- `descarga.css` - Estilos de la página de descargas
- `termynal.js` - Librería para la terminal animada

## Recursos:
- `img/` - Imágenes del sitio
- `svg/` - Iconos SVG

## Características:
- Páginas informativas sobre Cronux
- No requiere backend
- Funciona con archivos estáticos
- Enlaces a la comunidad dinámica
