Cronux-CRX: Control de versiones simple para todos

Cronux-CRX (Cronux Control de Revisiones eXtendido) es un sistema de control de versiones local diseñado para ser simple, rápido y efectivo. Funciona completamente offline, sin necesidad de repositorios remotos o configuraciones complejas.

crx new mi-proyecto ✓ Proyecto inicializado ID: 1.0 crx save -m "nueva funcionalidad" ✓ Versión guardada: 1.1 crx log 1.1 - nueva funcionalidad 1.0 - Proyecto inicializado crx restore 1.0 ✓ Restaurado a 1.0

Tu Primer Proyecto con Control de Versiones

Aprende los conceptos básicos del control de versiones con estos simples pasos. En minutos estarás organizando tus proyectos como un profesional.

1

Crear nuevo proyecto

Crea tu primer proyecto con control de versiones. Cronux configurará todo automáticamente para que puedas enfocarte en aprender los conceptos fundamentales.

Terminal
crx new mi-proyecto
Resultado:

✓ Proyecto inicializado
ID: 1.0

2

Guarda tu primera versión

Aprende a crear "snapshots" de tu trabajo. Cada vez que hagas cambios importantes, guarda una versión con un mensaje que te ayude a recordar qué hiciste.

Terminal
crx save -m "nueva funcionalidad"
Resultado:

Guardando versión...
✓ Versión guardada
ID: 1.1

3

Explora el historial

Descubre cómo visualizar la evolución de tu proyecto. El historial te muestra todos los cambios que has hecho a lo largo del tiempo.

Terminal
crx log
Resultado:

1.1 - nueva funcionalidad
1.0 - Proyecto inicializado

4

Restaura versiones anteriores

La magia del control de versiones: puedes volver a cualquier punto anterior de tu proyecto. Experimenta sin miedo, siempre puedes deshacer cambios.

Terminal
crx restore 1.0
Resultado:

✓ Restaurado a 1.0
