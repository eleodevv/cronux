Comandos Principales

Estos son los comandos esenciales que necesitas para usar Cronux-CRX. Haz clic en "Copiar" para usar cualquier comando directamente.

crx new

Básico

Crea un nuevo proyecto con control de versiones Cronux-CRX.

Sintaxis
crx new [nombre-proyecto]

Ejemplos:

crx new mi-proyecto Crea un proyecto llamado "mi-proyecto"
crx new tesis-universidad Crea un proyecto para tu tesis

crx save

Básico

Guarda una nueva versión del proyecto con un mensaje descriptivo.

Sintaxis
crx save -m "mensaje"

Ejemplos:

crx save -m "Primera versión" Guarda con un mensaje descriptivo
crx save -m "Capítulo 3 terminado" Ideal para proyectos académicos

crx log

Consulta

Muestra el historial completo de versiones del proyecto.

Sintaxis
crx log

Ejemplos:

crx log Muestra todas las versiones guardadas
crx log Perfecto para revisar tu progreso

crx restore

Navegación

Restaura el proyecto a una versión específica.

Sintaxis
crx restore [version]

Ejemplos:

crx restore 1.0 Restaura a la versión 1.0
crx restore 2.5 Restaura a la versión 2.5

crx status

Consulta

Muestra información del proyecto actual y archivos modificados.

Sintaxis
crx status

Ejemplos:

crx status Muestra el estado actual del proyecto
crx status Ideal para revisar cambios pendientes

crx --version

Información

Muestra la versión instalada de Cronux-CRX.

Sintaxis
crx --version

Resultado:

Cronux-CRX v0.1.0 Beta - Control de versiones local Muestra la versión actual instalada

crx help

Ayuda

Muestra la ayuda completa con todos los comandos disponibles.

Sintaxis
crx help

Ejemplos:

crx help Muestra todos los comandos disponibles
crx help Perfecto cuando necesitas recordar la sintaxis

Desinstalar

Sistema

Elimina completamente Cronux-CRX de tu sistema (macOS/Linux).

Sintaxis
sudo /usr/local/cronux/uninstall.sh

Nota:

sudo /usr/local/cronux/uninstall.sh Requiere permisos de administrador
